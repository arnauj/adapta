# Adapta

Aplicación para el Diseño de Actividades en Paneles Táctiles Activos.

## Descripción

Adapta es una herramienta diseñada para facilitar la creación y gestión de actividades educativas e interactivas adaptadas para su uso en paneles táctiles activos, mejorando la experiencia de aprendizaje mediante recursos visuales y manipulativos.

## Características

- **Gestión de actividades**: Interfaz intuitiva para organizar y crear actividades.
- **Soporte multimedia**: Carga y visualización de documentos PDF e imágenes.
- **Herramientas de dibujo**: Interfaz de dibujo integrada para actividades prácticas.
- **Personalización**: Ajustes visuales y configuraciones para adaptar la experiencia a diferentes necesidades.

## Tecnologías Utilizadas

- **Flutter**: Framework principal para el desarrollo de la interfaz multiplataforma.
- **Dart**: Lenguaje de programación base.
- **Syncfusion Flutter PDF Viewer**: Visualización avanzada de documentos PDF.
- **Scribble**: Herramienta de dibujo sobre lienzo.
- **Provider / StateNotifier**: Gestión de estado de la aplicación.
- **File Picker**: Selección de archivos desde el dispositivo.
- **Url Launcher**: Integración con enlaces externos.

## Configuración y Desarrollo

1. Asegúrate de tener instalado el SDK de Flutter y Dart.
2. Clona el repositorio.
3. Instala las dependencias: `flutter pub get`.
4. Ejecuta la aplicación: `flutter run`.

## Estructura del Proyecto

- `lib/`: Contiene el código fuente principal de la aplicación.
  - `screens/`: Vistas y pantallas de la aplicación.
  - `widgets/`: Componentes reutilizables de UI.
  - `providers/`: Lógica de gestión de estado.
  - `models/`: Modelos de datos.
- `assets/`: Recursos estáticos (imágenes, iconos).
- `web/`: Configuración para despliegue web.
